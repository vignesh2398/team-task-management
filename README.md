# Team Task Management App

A full stack task management application built using:

- Frontend: React + Vite
- Backend: Node.js + Express + TypeScript
- State Management: Zustand
- Styling: Bootstrap

---

# Project Structure

project/
│
├── backend/
│
└── frontend/

---

# Backend Setup (TypeScript + Express)

## 1. Navigate to backend

```bash
cd backend
npm i
npm run build
npm start

Backend Runs on http://localhost:3000

## 2. Navigate to frontend

cd frontend
npm i
npm run dev
